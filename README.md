# Particle Portfolio

Interactive 3D developer portfolio built with Three.js — features an animated 3D hand model surrounded by dynamic particle effects, serving as a creative showcase of personal projects and skills.

**Live:** [nvneth.com](https://nvneth.com)

## What It Does

- Renders a 3D hand model (`.glb`) with Three.js and WebGL
- Generates animated particle systems around the model
- Loads projects from `PROJECTS.md` and displays them in a minimal UI
- Fully responsive, single-page experience

## Tech Stack

| Layer | Tech |
|-------|------|
| Bundler | Vite |
| 3D Rendering | Three.js |
| Language | JavaScript |
| Deployment | Static hosting |

## Practical Use

Portfolio site that replaces a standard resume page with an interactive 3D experience — communicates technical creativity to potential employers or collaborators at first glance.

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
