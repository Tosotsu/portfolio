Place your ambient audio file here as: ambient.mp3

Recommended: a long (~5-10 min) ambient/drone track at 128kbps.
The AudioManager will loop it seamlessly.

If this file is missing, the mute button is automatically disabled
and no audio error is thrown — the site functions fully without audio.

Royalty-free ambient sources:
- freesound.org (search "ambient drone")
- zapsplat.com
- pixabay.com/music
